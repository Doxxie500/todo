## 📌 Overview

The **Todo Web App** allows users to:

- Add tasks
- Mark tasks as completed
- Delete tasks
- Stay organized daily

It demonstrates core frontend concepts like **DOM manipulation**, **event handling**, and **state management** using JavaScript.

---

## ✨ Features

✅ Add new tasks  
✅ Mark tasks as completed  
✅ Delete tasks  
✅ Real-time UI updates  
✅ Clean and responsive design  
✅ Simple and user-friendly interface  
✅ Lightweight and fast
